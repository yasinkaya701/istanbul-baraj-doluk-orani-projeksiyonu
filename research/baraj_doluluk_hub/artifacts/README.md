# Artifacts

This folder tracks delivery-facing and model-facing outputs that matter beyond
one-off experiments.

Current control files:

- `artifact_catalog.csv`
  canonical machine-readable artifact register
- `ARTIFACT_INDEX.md`
  human-readable grouped index generated from the catalog

Rule:

If an output matters for presentation, validation, reporting, or operational
follow-up, it should appear in `artifact_catalog.csv`.
